Atenção sobre compartilhamento

Há duas formas principais de embutir um relatório Power BI:

  1. Publish to web — deixa o relatório público (qualquer um com o link pode acessar). Fácil para compartilhar no GitHub Pages.
  2. Embed seguro (organização) — exige autenticação / token (Power BI Pro / service principal). Recomendado para dados sensíveis.

Se o relatório exigir login, o iframe não carregará para usuários que não tenham permissão. Neste caso use Publish to web (apenas se o conteúdo for público).
